Overview

Fault Locator Pro is a Progressive Web App (PWA) designed for electrical
and low-voltage technicians to estimate the distance to a short circuit
in a two-conductor solid copper cable.
Fault Locator Pro


The app calculates the estimated fault location by using:

-   Wire gauge (AWG)
-   Measured loop resistance (ohms)
-   Estimated wire temperature
-   Optional total cable length

The application compensates for the change in copper resistance caused
by temperature and estimates the distance from the test location to the
fault.

--------------------------------------------------------------------------

Features

-   Supports 14, 16, 18, 22, and 24 AWG solid copper wire
-   Temperature compensation based on copper resistance
-   Distance-to-fault calculation
-   Optional cable length for visual fault position
-   Save job information
-   Job history stored on the device
-   GPS location capture (optional)
-   Copyable field report
-   Offline operation after installation
-   Optimized for iPhone Home Screen installation

------------------------------------------------------------------------

Typical Workflow

1.  De-energize and isolate the cable.
2.  Measure loop resistance across the two conductors with a digital
    multimeter.
3.  Select the wire gauge.
4.  Enter the measured resistance.
5.  Enter the estimated wire temperature.
6.  Optionally enter the total cable length.
7.  Review the estimated distance to the short.

------------------------------------------------------------------------

Installing on iPhone

1.  Open the GitHub Pages URL in Safari.
2.  Tap Share.
3.  Tap Add to Home Screen.
4.  Launch the app from the Home Screen.

Once installed, the app functions like a native iPhone application and
continues to work offline after the necessary files have been cached.

------------------------------------------------------------------------
Disclaimer!

This application provides an engineering estimate based on standard
resistance values for solid copper conductors. Actual fault locations
can vary because of splices, corrosion, poor connections, partial
shorts, parallel paths, inaccurate temperature estimates, or incorrect
wire gauge selection.

Always verify calculations using accepted electrical testing procedures
and follow all applicable electrical safety practices.
